Shiny App Tutorial


All resources from this project is credited. Please contact me if there is any question.


1. Download R Studio

Note: Download R can be tricky. Most of the time users code in R studio that can be found in the link below. The letter "R" itself means something else.

Here!


2. Open R Studio and Open a New R Script


3. Find console and type in the following code:

install.packages("shiny")

Note: console locates at the lower half of your left side of the screen. This step allows you to install shiny package.


4. Type in the following in the empty R script:

library(shiny)

Note: Remember that you always need to call a package after installation.


There is a lot to do with shiny. Example code can be found here.


5. Import dataset

Sample code:

read.table("where_you_store_your_data")
read.xlsx("where_you_store_your_data")

Other ways to import data

Note: The dataset I used in this project is from datafest 2019.


6. install shinydashboard

install.packages("shinydashboard")

library(shinydashboard)

7. Introduce leaflet in R
